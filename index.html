リハ計算アプリ一覧

現場で使いやすいツール集。iPhoneの場合は各アプリをSafariで開いて「共有 → ホーム画面に追加」して使えます。

10m歩行・HHD（筋力→トルク） 計算
・10m歩行（時間・歩数） → 速度/歩幅/ケイデンス
・HHD (N) × モーメントアーム (cm) → Nm / Nm/kg (基準値表示)
開く
BBS入力 (脳卒中 / 大腿骨近位部骨折) 評価
・14項目（0〜4点 / 最大56点）タップ入力
・シート切替、cutoff判定、前回との差 (MDC/MCID)
開く
EMG歩行解析（スマホ対応） EMG
・CSVを選択（HS基準）
・筋活動のみでHS推定（動画目視HSでガード）
・%peak → 歩行周期0-100%正規化 → 平均±SD
・HS一覧CSV / プロットPNG保存
開く
SPPB (Short Physical Performance Battery) 評価
・バランス、歩行速度、5回椅子立ち上がり
・ストップウォッチ機能での自動計測
・合計点数の自動計算・結果コピー機能
開く
FMA (Fugl-Meyer Assessment) 評価
・上肢(66点)/下肢(34点)の運動スコア計算
・重症度分類(軽度/中等度/重度)の自動判定
・MDC/MCIDの臨床指標を表示
開く
